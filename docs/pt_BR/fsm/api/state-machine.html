public class StateMachine : MonoBehaviour {}

apenas para feedback visual no inspetor

status

O status atual da máquina

protected string

currentStateName

Um feedback visual para o inspetor

protected State

defaultState

O estado padrão para ser definido usando o inspetor

recognizedStates

Feedback visual

states

Lista de estados anexados ao GameObject da máquina.
Será carregado durante o StateMachine.SetUp()

Métodos

public virtual void

SetUp()

Chamado pelo Ator para inicializar a máquina

Argumentos: Actor actor”

protected virtual void

LoadStates()

Carrega todos os estados anexados ao GameObject da máquina.

public virtual void

StartMachine()

Se a máquina está pronta, define o currentState como o estado padrão e o status como On

public virtual void

Resume()

status = On

public virtual void

Pause()

status = Pause

public virtual void

Stop()

status = Off

public virtual void

ChangeState()

Define o currentState como o estado fornecido

Argumentos: State state”

protected virtual void

EvaluateNextState()

Caso alguma condição das transições transição do currentState seja atendida,
a máquina irá transitar para o estado alvo

protected virtual State

ConditionMet()

Verifica a StateTransition.Condition() de todas as transições do currentState. Retorna null se nenhuma condição for atendida.

public virtual void

SetDefaultState()

Sets the defaultState

Argumentos: T onde T : State

public virtual void

Tick()

Chamado a cada Actor MonoBehaviour.Update(). Força a máquina a
avaliar se deve mudar o currentState e chama
o currentState State.Tick()

public virtual void

LateTick()

Chamado a cada Actor MonoBehaviour.Update(). Força a máquina a
avaliar se deve mudar o currentState e chama
o currentState State.Tick()

public virtual void

FixedTick()

Chamado a cada Actor MonoBehaviour.FixedUpdate(). Força a máquina a
avaliar se deve mudar o currentState e chama
o currentState State.FixedTick()

Eventos

public UnityEvent<State>

StateChanged

Invocado toda vez que a máquina muda o currentState
